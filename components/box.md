# Box

Layout container component.

## Basic Usage

::: tabs vue3 vue2 react
@tab vue3

```vue
<SBox padding="20px">
  <div>Content</div>
</SBox>
```

@tab vue2

```vue
<SBox padding="20px">
  <div>Content</div>
</SBox>
```

@tab react

```tsx
<SBox padding="20px">
  <div>Content</div>
</SBox>
```
:::

## Props

### Vue3 / Vue2 / React

| Name | Description | Type | Default |
|------|-------------|------|---------|
| padding | Inner padding | `string` | `'0'` |
| margin | Outer margin | `string` | `'0'` |
| border | Border style | `string` | - |
