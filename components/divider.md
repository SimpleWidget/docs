# Divider

Content separator line.

## Basic Usage

::: details Vue3

```vue
<SDivider />
<SDivider direction="vertical" />
```

:::

::: details Vue2

```vue
<SDivider />
<SDivider direction="vertical" />
```

:::

::: details React

```tsx
<SDivider />
<SDivider direction="vertical" />
```

:::

## Props

| Name | Description | Type | Default |
|------|-------------|------|---------|
| direction | Separator direction | `'horizontal' \| 'vertical'` | `'horizontal'` |
| contentPosition | Content position | `'left' \| 'center' \| 'right'` | `'center'` |
