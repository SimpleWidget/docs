# BackTop

Scroll back to top of page.

## Basic Usage

::: details Vue3

```vue
<SBackTop />
```

:::

::: details Vue2

```vue
<SBackTop />
```

:::

::: details React

```tsx
<SBackTop />
```

:::

## Props

| Name | Description | Type | Default |
|------|-------------|------|---------|
| target | Scroll container selector | `string` | - |
| visibilityHeight | Show when scroll distance | `number` | `200` |
| duration | Scroll duration (ms) | `number` | `400` |
