# Empty

Empty state display.

## Basic Usage

::: details Vue3

```vue
<SEmpty description="No data" />
```

:::

::: details Vue2

```vue
<SEmpty description="No data" />
```

:::

::: details React

```tsx
<SEmpty description="No data" />
```

:::

## Props

| Name | Description | Type | Default |
|------|-------------|------|---------|
| image | Custom image URL | `string` | - |
| description | Description text | `string` | `'No data'` |
